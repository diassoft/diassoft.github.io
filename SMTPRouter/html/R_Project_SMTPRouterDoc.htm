Namespaces

Namespaces
NamespaceDescription
SMTPRouter
The SMTPRouter namespace contains the types that allow the creation of a Listener and a Router to listen to SMTP Messages and Route them.
SMTPRouter.Models
The SMTPRouter.Models namespace contains the types that represent the data used by the SMTPRouter classes. All models are compatible with Mvvm implementations.
SMTPRouter.Windows.Configuration
The SMTPRouter.Windows.Configuratoin namespace contains the types that allow the system to parse a Configuration File (such as App.Config or Web.Config) in a meaningful way.
